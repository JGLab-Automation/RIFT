2.1.1.1  list rw-certificate:certificate
  Description
    Server certificate configuration. If this certificate is also
    used as client certificate for client authentication, then the certificate
    must have been created with that key usage.
  Leaves
    name: name, type: STRING, description: An identifier for the certificate
    name: certificate-type, type: ENUM, description: The type of certificate, Internal/External.
    Internal certificates are strictly for internal Riftware services which
    are not exposed.
    External certificates are for services which are exposed to Northbound.
    name: server-certificate, type: STRING, description: The server certificate to be installed. Only PEM format accepted.
    name: private-key, type: STRING, description: Private key for the corresponding server certificate. Only PEM format
    accepted. The private key must be unencrypted.
    name: ca-certificate, type: STRING, description: The CA certificate or certificate chain for the corresponding
    server certificate. Only PEM format accepted.
    NOTE: The CA installed here is Rift specific hence not installed
    on the system CA bundle path.
  XPath Path
